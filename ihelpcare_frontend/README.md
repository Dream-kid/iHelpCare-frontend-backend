## Getting Started

Commands, to exciting & controlling application:

```bash
npm run dev # start dev server
npm run build # application prod build
npm run start # start prod server
npm run format # formatting source code
npm run lint # code linting check
npm run analyze # analyze bundle size
```

Open [http://localhost:3041](http://localhost:3041) with your browser to see the result.

You can start editing the page by modifying `src/app/[lang]/(root)/page.jsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
